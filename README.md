# Олимпиада Mark.Online — руководство пользователя

Этот репозиторий содержит одностраничную игру «Олимпиада». Проект полностью статический: HTML, CSS, JavaScript и файлы конфигурации находятся в корне. Игру можно запускать локально или размещать на любом статическом хостинге.

Техническая документация для разработчиков вынесена в [`TECHNICAL.md`](TECHNICAL.md).

## Быстрый старт

1. Откройте `index.html` в браузере или запустите локальный сервер:
   ```bash
   python -m http.server 8000
   ```
2. Перейдите на `http://localhost:8000`.
3. На стартовом экране выберите сезон, затем программу.
4. После выбора откроется игровое поле и панель управления.

## Как играть

1. Выберите сезон и программу.
2. Наберите команды в программе, нажимая на доступные кнопки.
3. Запустите программу или выполняйте команды пошагово.
4. Собирайте героев и откройте ящик, чтобы завершить сценарий.

## Управление

- **Запустить** — выполняет всю программу.
- **Шаг** — выполняет следующую команду.
- **Очистить** — очищает программу.
- **Удалить** — удаляет последнюю команду.
- **Сбросить прогресс** — обнуляет прохождение для выбранного варианта.

## Полный экран

- Кнопка **«Во весь экран»** скрывает панели и растягивает поле на всю ширину.
- В полноэкранном режиме поверх поля появляется кнопка **«Вернуться»** для выхода.

## Доступные варианты

Выбор двухэтапный:

1. **Сезон**: Зимняя или Весенняя Олимпиада.
2. **Программа**:
   - Зима: `J2`, `J3`, `J4`
   - Весна: `J3`, `J4`, `P3`

Итого доступно шесть вариантов: `winter-j2`, `winter-j3`, `winter-j4`, `spring-j3`, `spring-j4`, `spring-p3`.

## Советы ведущему

- Для смены сложности изменяйте количество учеников и баллы группы — это влияет на доступность команд.
- При необходимости используйте кнопку «Сбросить прогресс» перед новой группой.
