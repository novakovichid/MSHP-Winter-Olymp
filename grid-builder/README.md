# Редактор сетки Олимпиады

Этот раздел содержит отдельную WYSIWYG-страницу для сборки игрового поля. Она не включена в навигацию основного сайта и используется как вспомогательный инструмент для подготовки данных `board.json`.

## Как открыть

Запустите статический сервер из корня проекта и перейдите по адресу:

```
http://localhost:8000/grid-builder/
```

Страница загружает `board.json` (и сопровождающие ресурсы) напрямую, поэтому всегда отображает актуальные герои, планеты, ассеты и механики.

## Как работать

1. Выберите размер сетки.
2. Выберите инструмент (путь, старт, объект, герой, планета, ластик).
3. Нажимайте на клетки поля для размещения элементов.
4. При необходимости загрузите готовый вариант из `board.json` и используйте его как шаблон.
5. Скопируйте JSON-фрагмент из правой панели и вставьте его в `board.json`.

## Что формируется

Редактор генерирует JSON-фрагмент с ключами:

- `grid` (размеры, путь, старт и объекты)
- `heroes`
- `planets`
- `assets` (переопределения ассетов)
- `overseer`

Можно копировать весь фрагмент как основу для нового варианта.
